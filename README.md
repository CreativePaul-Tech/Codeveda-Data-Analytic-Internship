Project Overview
This repository contains the code and datasets for the tasks completed during my Data Analytics Internship at Codveda. The projects demonstrate a progression from fundamental data handling to advanced machine learning and natural language processing techniques.
Tools & Technologies
#Programming: Python  
#Data Manipulation: Pandas  
#Machine Learning: Scikit-Learn  
#Natural Language Processing: NLTK, TextBlob  
#Data Visualization: Matplotlib, Seaborn, WordCloud  
Tasks Completed
#Level 1: Data Cleaning & Exploratory Data Analysis (EDA)
Processed the Iris dataset by identifying missing values and removing duplicates. Conducted EDA by generating distribution histograms, feature boxplots, and correlation heatmaps to uncover relationships between variables.  
#Level 2: Regression & Time Series Analysis
Developed a Linear Regression model to predict stock closing prices based on opening market prices. Applied Time Series Analysis to Apple (AAPL) stock data, calculating and visualizing a 5-day moving average to smooth price trends over time.  
#Level 3: Predictive Modeling & Sentiment Analysis
Built a customer churn prediction model, comparing Logistic Regression, Decision Trees, and Random Forest algorithms, ultimately optimizing the Random Forest model using GridSearchCV to achieve over 94% accuracy. Performed Natural Language Processing (NLP) to categorize text data into Positive, Negative, and Neutral sentiments, visualizing the most frequent terms using a custom word cloud.
